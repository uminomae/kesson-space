# kesson-space

> pjdhiroの主観世界を追体験する3D空間

神経現象学的な意識モデルを、暗闇と夢のような視覚体験として表現する。

## コンセプト

- **意識の4層**: 内受容感覚 → 予測誤差 → F-O評価 → Withhold
- **創造の5段階**: 場 → 波 → 縁 → 渦 → 束（海のメタファー）
- **忍的雰囲気**: 秘すれば花、正心、未発の中

詳細: [docs/CONCEPT.md](docs/CONCEPT.md)

## 技術スタック

- GitHub Pages
- Vanilla JS + Three.js
- Git as Database（YAML）
