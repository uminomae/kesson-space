セッション初期化を実行してください。以下の手順を順番に実行し、結果をまとめて報告してください。

## 手順

### 1. 現在の状態を確認
`docs/CURRENT.md` を読んで、最終セッション番号・実施内容・次タスクを把握する。

### 2. 環境確認
- `pwd` で現在のワークツリーを確認（CLAUDE.md §2テーブルと照合）
- `git status` で作業ツリーの状態を確認
- `git log --oneline -5` で直近のコミットを確認

### 3. タスク確認
`docs/TODO.md` を読んで、P0/P1 タスクを確認する。

### 4. 状態レポートを出力

以下のフォーマットでユーザーに報告:

```
## セッション初期化レポート

### 環境
- ワークツリー: [名前] (`pwd`の結果)
- ブランチ: [ブランチ名]
- 作業ツリー: [clean / dirty]

### 前回セッション
- #[番号] ([日付]): [概要]

### 推奨タスク
- [P0/P1タスクのリスト]

### 負荷判定
- [🟢低 / 🟡中 / 🔴高] — [理由]
```
