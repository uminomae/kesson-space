# E10: 安定した関係が、待てる力を育てる

## 理論的背景

Bowlbyの愛着理論に基づく：
- 安定型愛着 → 安全基地 → 探索行動 → 抱持能力
- 不安定型愛着 → 探索制限 → 即時反応傾向

## 概念の精緻化（ISS-36）

### 内在化された関係

**なぜこの精緻化が必要か**:

「安定した関係」は外的・物理的関係を想起させるが、pjdhiroの意図（S18）は内在化された関係を含む。

**pjdhiro発言（S18）**:
> "内面に" 信頼できる誰かがいるから。物理的にいなくても、死んだ人でも大丈夫です。小さい頃のお母さんの記憶でも大丈夫です。

**誤解されやすいポイント**:
- ❌「物理的に誰かが側にいる必要がある」
- ✅「心の中に内在化された関係があればよい」

**関連概念**: CN-001（内在化された関係）を参照

## 抱持能力との関係

```
安定した関係（愛着）
    ↓
内的作業モデル（安全基地の内在化）
    ↓
信頼軸の安定（関係への信頼）
    ↓
生存軸の制御（脅威反応の緩和）
    ↓
抱持能力の発達
```
