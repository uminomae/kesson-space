# E06: すぐに解決しないことが、美と創造の条件である

## 位置づけ

E05「美とは、欠損を保持することで生じる」の帰結・具体化

## 群盲象チェック

| 領域 | 表現 | 検証状態 |
|------|------|---------|
| 美学 | 美的体験は解決を保留する | ✅ |
| 創造性研究 | インキュベーション期間の必要性 | ✅ |
| 精神分析 | 早期解釈の危険性 | ✅ |
| 禅 | 不立文字、教外別伝 | ✅ |

## バリエーション

| 読者 | 表現 |
|------|------|
| 研究者 | 解決の遅延が創造的転換の必要条件 |
| 芸術家 | 作品は「完成しない」ことで美しい |
| 経営者 | 拙速な意思決定は選択肢を狭める |
| 教育者 | 答えをすぐに教えないことが学びを深める |
| 一般 | 急いで答えを出さない方がいい時がある |
