# E04: AIは予測誤差を計算するが、「欠損」を経験しない

## 論点

| 観点 | AI | 人間 |
|------|----|----|
| 予測誤差 | 計算する | 計算する |
| 主観的経験 | なし | 「欠け」として経験 |
| 身体性 | なし | 内受容感覚（Layer 0） |
| 価値評価 | 設計された目的関数 | F-O座標系で構成 |

## 群盲象チェック

| 領域 | 表現 | 検証状態 |
|------|------|---------|
| 神経科学 | 内受容感覚の有無 | ✅ |
| 現象学 | クオリアの有無 | ✅ |
| AI研究 | シンボルグラウンディング問題 | ✅ |

## バリエーション

| 読者 | 表現 |
|------|------|
| 研究者 | 予測誤差の主観的経験は身体性に依存する |
| エンジニア | AIにはLoss関数があるが、それを「痛い」と感じない |
| 経営者 | AIは「困る」ことができない |
| 一般 | AIは間違いを直すけど、悔しいとは思わない |
