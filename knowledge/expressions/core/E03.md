# E03: 創造は、際（境界）に留まることを経て生じる

## 群盲象チェック

| 領域 | 表現 | 検証状態 |
|------|------|---------|
| 数学（Julia集合） | 境界上の軌道を歩く | ✅ |
| 物理（QFT） | 真空ゆらぎに留まる | ✅ |
| 神経科学 | 準安定状態の維持 | ✅ |
| 欠損駆動 | 抱持による保持 | ✅ |

## 5段階への位置づけ

```
場（Field）: 完全な未分化
波（Wave）: 揺れ始め
**縁（Relation）: 準安定状態そのもの** ← 「際」はここ
渦（Spinor）: 個として立ち上がる
束（Bundle）: 方向を持つ
```

## バリエーション

| 読者 | 表現 |
|------|------|

| 物理学者 | 場の励起状態を維持すること |
| 一般 | どっちつかずの状態に耐えること |
