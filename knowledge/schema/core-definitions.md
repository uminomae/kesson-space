## D1: 欠損駆動思考（Kesson-Driven Thinking）

### 定義
棄却される誤差を、問いとして拾う態度

### 位置づけ
- レベル: L2（態度・哲学）
- 上位: プロジェクトデザイン（L1）
- 下位: 欠損駆動開発（L3）、各技法（L4）

### 補足
- 「思考」は態度・姿勢を指す
- 具体的な方法論は「欠損駆動開発（L3）」で扱う

## D2: 欠損（Kesson）

### 定義
予想と現実の誤差を、意識が「欠け」として捉えた主観的経験

### 構成要素
1. **予測誤差**（客観）: 予想と現実のズレ [P]
2. **主観的経験**（主観）: 「欠け」として意識に上る [M]

### 5類型
| 類型 | 内容 |
|------|------|
| 観測欠損 | 事実と予測のズレ |
| 主体欠損 | 自己像と現実のズレ |
| 正当化欠損 | 行動と価値観のズレ |
| 一貫性欠損 | 信念間の矛盾 |
| 意味欠損 | 意味の喪失・空虚 |

### 神経基盤 [P]
- Layer 1（予測-誤差ループ）で検出
- 精度閾値を超えると意識化

### AIとの対比
- AI: 予測誤差を**計算**する
- 人間: 予測誤差を**経験**する（欠損として）

### 記述水準の区別

欠損（D2）は二つの記述水準を持つ。

| 水準 | 記述 | 例 |
|------|------|-----|
| 3人称（計算論的） | 予測誤差信号が精度閾値を超えて伝播する | 「誤差信号ε が precision-weighted で上位層に送られる」 |
| 1人称（現象学的） | 予想と現実の誤差が「欠け」として体験される | 「何かが足りない、という感覚がある」 |

本理論のD2は1人称記述を主語とする。3人称の計算論的記述（予測符号化、自由エネルギー原理）は参照枠であり、D2そのものではない。「意識化された予測誤差」と「欠けとして体験された予測誤差」は同じ現象の異なる記述水準であるが、本理論が問題にするのは後者である。

**検討経緯**: REVIEW-006 §1（C1: D2定義の情報落ち問題）、§6 Task 4

## D3: 抱持（旧称: Withhold）

### 定義
反射的に処理せず、誤差を問いとして抱え保つ機能

### 位置づけ
- Layer 3（意識の4層モデル）
- 「意のゲート」の機能的記述

### 対比
| 概念 | 機能 |
|------|------|
| 反応抑制 | 行動を**止める** |
| 抱持 | 行動準備を**保持しつつ再評価** |

### 関連概念（群盲象チェック）
- Bion: Container（β要素の保持）
- Klein: PSに留まる耐性
- 世阿弥: 秘すれば花
- 葉隠: 忍ぶ恋
- 中庸: 未発の中

### 神経基盤 [P]
- dlPFC（背外側前頭前野）
- ACC（前帯状皮質）

### 成立条件（D3-a）

抱持は個人の意志力ではなく、複数の条件が層的に支えることで成立する機能である。

| 層 | 成立条件 | 具体例 | 根拠 |
|----|----------|--------|------|
| L0 | 生理的余裕（恒常性維持） | 睡眠、栄養、身体的安全、ワーキングメモリ容量 | [P] 神経科学 |
| L0-L1 | 自律神経調整 | 安全な関係（E10）、身体的修練（瞑想、武道、呼吸法）、環境設計 | [P] Porges, Bowlby; [M] 禅、ヨーガ |
| L1-L2 | 外部Container（構造・制度） | 公案、茶道の作法、分析の設定、アンドン、学問のディシプリン | [P] Bion後期; [M] 世阿弥、茶道 |
| L2-L3 | 認知的枠組み（メタ認知） | epochē（判断保留）、覚悟（葉隠）、ネガティブ・ケイパビリティ | [M] ストア、Keats→Bion、ハイデガー |

安定した関係（E10）はL0-L1経路の中で最も発達的に根源的なものだが、唯一の経路ではない。

**検討経緯**: `docs/decision-log/2026-02/LOG-20260205-001`

### 層間再入力（生存-信頼循環モデル）

抱持（L3）の出力はL1-L2に再入力され、欠損の意味が更新される。これにより「わからない」の質が変容する循環構造をなす。

> 抱持は、生存軸Containment（生理的安全の確保）のもとで信頼軸α変換（問いとしての保持と変換）が作動し、その出力がL1-L2に再入力されることで欠損の意味が更新される循環プロセスである。

この循環は計算論的には、L3がpolicy precisionを下げつつ予測誤差への重み付けを維持することで、L1-L2の内部モデルが更新される過程として参照できる。一時停止→再開（解釈A）でも意図形成途中（解釈B）でもなく、L3→L1-L2→L3…の循環が「問いとして保持する」の構造をなす（解釈C）。

**検討経緯**: `docs/decision-log/2026-02/LOG-20260205-002`、REVIEW-006 §2（解釈C策定）

### 計算論的参照枠（暫定）

以下は抱持の構造を能動推論（Active Inference）の語彙で参照する試みである。本理論はこの参照枠に還元されない。

#### precision 3層

| 層 | 概念 | 抱持時の操作（暫定） |
|----|------|------------------------|
| sensory precision | 感覚入力の信頼度重み付け | 変動あり（局面依存） |
| prior precision | モデル予測の信頼度重み付け | 低下（既存モデルを緩める） |
| policy precision | 行動方策の信頼度重み付け | **低下**（自動行動の抑制） |

加えて、本理論では予測誤差そのものへの精度重み付けが上昇する操作を想定する（仮称「error precision」）。これはAIF標準語彙ではなく、本理論の暫定表現である。

#### 既存概念との計算論的対比

| 概念 | precision操作 | 時間スケール | 目的関数 |
|------|-------------|-------------|----------|
| 反応抑制 | policy↓ | 瞬時 | 誤反応回避 |
| 回避/フリーズ | sensory↓ | 持続 | 脅威最小化 |
| 認知的再評価 | prior調整 | 秒〜分 | 情動制御 |
| 不確実性耐性 | — | 特性的 | — |
| **抱持（本理論）** | **policy↓, error precision↑（仮置き）** | **秒〜（循環的）** | **欠損の意味更新（L1-L2再入力）** |

抱持の独自性は、行動を止めることではなく、誤差（欠損）を保持しその意味を循環的に更新する点にある。反応抑制が「止める」、認知的再評価が「捉え直す」のに対し、抱持は「問いとして留める」。

**検討経緯**: REVIEW-006 §2（解釈C）、§6（DRAFT-GPT-007レビュー）

## D4: 情動の構成

### 定義
欠損が生存軸（生存）と信頼軸（愛）で評価され、情動として構成されるプロセス

### 生存-信頼座標系

| 軸 | 正式名 | 評価内容 | 神経基盤 |
|----|--------|---------|---------| 
| 生存軸 | Fear/Fight | 生存・脅威 | 扁桃体 |
| 信頼軸 | Others/Attachment | 愛着・所属 | vmPFC |

### プロセス
```
欠損（D2）の検出
    ↓
生存軸評価: 「脅威か？」
信頼軸評価: 「関係に影響するか？」
    ↓
情動として構成
    ↓
抱持（D3）または即時反応
```

### 理論的背景 [P]
- Barrett (2017): 構成主義的情動理論
- 情動 = 内受容誤差 + 文脈 + 評価

## 定義間の関係

```
D1（欠損駆動思考）= 態度
    │
    │ この態度で扱う対象が
    ↓
D2（欠損）= 主観的経験
    │
    │ 欠損を評価するのが
    ↓
D4（情動の構成）= 生存-信頼評価
    │
    │ 評価後に即時反応せず保持するのが
    ↓
D3（抱持）= 保持機能
```
