# 秘される花 - Kesson Space

**バージョン**: 0.4
**更新日**: 2026-02-10

## コンセプト

> pjdhiroの主観世界を追体験する3D空間

神経現象学的な意識モデルを、暗闘と夢のような視覚体験として表現する。
東洋思想の西洋科学的論証。忍的な雰囲気（露骨にしない）。

---

## 主客の配置

```
【上空】意識（ユーザーの視点）= 4層の主体
    │
    │ 見下ろす
    ▼
┌─────────────────────────────────┐
│ 闇（潜在的な誤差の海）          │
│ 空間全体が呼吸している         │
│                                 │
│  ◌ ← 光が滲み出る（欠損）      │
│                                 │
├─────────────────────────────────┤
│ ～～ 水面（縁）～～              │
│ フラクタル的な波紋              │
├─────────────────────────────────┤
│ 【水面上】存在の現前            │
│ 世界、宇宙、創造されたもの     │
│ クリック → ここに「降りる」     │
└─────────────────────────────────┘
```

- **水面上** = 存在の現前（客観世界、創造されたもの）
- **水面下〜闇** = 創造の深層
- **上空（ユーザー）** = 意識（4層の主体）

## 理論的基盤

### コア定義（D1-D4）

| ID | 定義 | 性質 | 3D表現 |
|----|------|------|--------|
| D1 | 棄却される誤差を、問いとして拾う態度 | 態度（メタ） | 空間全体の性質。完全な虚無ではなく、潜在的なものが満ちている |
| D2 | 予想と現実の誤差を、意識が「欠け」として捉えた主観的経験 | 経験 | 光の生成。闇から滲み出る、輪郭はJulia集合的 |
| D3 | 反射的に処理せず、誤差を問いとして保持する機能 | 機能 | 光の状態。消えずに留まっている、呼吸するように明滅 |
| D4 | 欠損がF軸（生存）とO軸（愛）で評価され、情動として構成されるプロセス | プロセス | 光の質感。色、動きの速さ、粒子の密度 |

### 意識の4層（M1）

| Layer | 概念 | ユーザーが「体験する」表現 |
|-------|------|--------------------------|
| L0 | 内受容感覚（身体の「今」） | 画面全体の呼吸（収縮拡張）、脈動、温度変化、微細な粒子 |
| L1 | 予測誤差→欠損の生成 | 闇から光が滲み出る。輪郭はJulia集合的、夢のように曖昧 |
| L2 | F-O評価（情動価の付与） | 光に色がつく、振る舞いが変わる |
| L3 | Withhold（保持） | 光が安定して「留まる」 |

**意のゲート**（L2-L3間）は意識の内部構造。空間に配置するものではない。

### 創造の5段階（M2）= 海

| 段階 | 名称 | 構造 | 海の表現 |
|------|------|------|---------|
| 1 | 場 | 無（未分化） | 深海の闇 |
| 2 | 波 | ゆれ・対立 | 深層の波動 |
| 3 | 縁 | 境界・関係 | 水面（フラクタル的波紋） |
| 4 | 渦 | 個・立ち上がり | 水面上の渦 |
| 5 | 束 | 方向 | 流れ・潮流、存在の現前 |

## 視覚表現

### 全体の質感（夢的・神経現象学的）

- エッジがはっきりしない
- 形が変化し続ける
- 色が滲む
- 粒子感（解像度が低い）

### 忍的雰囲気（露骨にしない）

- 墨色、深い藍、闇
- 差し色：月白、蛍光
- 霧、煙、水面の揺らぎ
- 静寂、微動

### 内受容感覚（L0）の表現

| 要素 | 表現 | 根拠 |
|------|------|------|
| 呼吸 | 空間全体がゆっくり収縮・拡張 | 最も基本的な内受容 |
| 脈動 | 低周波の波が画面を横切る | 心拍、血流 |
| 温度 | 色温度の微かな変化 | 体温感覚 |
| 粒子 | 空気中の微細な粒子が漂う | 身体内部の「充満感」 |
| 霧 | 呼気のような、吐き出される霧 | 呼吸の可視化 |

### 光（欠損）の表現

| 要素 | 表現 |
|------|------|
| 生成 | 闇から滲み出る |
| 輪郭 | Julia集合的、フラクタル、曖昧 |
| 形 | 不完全（欠けた円、揺らぐ形） |
| 色 | F軸=暖色系の緊張 / O軸=寒色系の安らぎ |
| 動き | F軸=速い / O軸=穏やか |
| 保持状態 | 安定して漂う、呼吸するように明滅 |

### シルエット（オプション）

- ヒーロー画像のような人型
- ユーザー自身の「意識の身体」
- 輪郭がJulia集合的に揺らぐ

### 水面

- フラクタル的な波紋
- 向こう側（水面上）が見える
- 境界としての「縁」

---

## 体験の流れ

1. **入場**: 上空から闇を見下ろす
2. **感じる**: 空間全体が呼吸している（L0）
3. **見える**: 闇から光が滲み出る（L1→L2→L3）
4. **近づく**: 光の色と動きを観察
5. **クリック**: 水面を通過し、存在の現前（詳細ページ）へ降りる
