## 問題

現在の `scripts/blender-x-logo.py` は2本の回転した直方体を重ねているだけで、
X社（旧Twitter）の正しいロゴ形状にならない。端が閉じていない線になる。

### 現在の間違った方式
- 右上がりの太い直方体（実線バー）
- 左上がりの中抜き直方体（ブーリアンでくり抜き）
- 2つを結合 → 水平カット

### X 社ロゴの正しい形状
X社のロゴは**2本のバーの単純な交差ではない**。以下の特徴を持つ:

1. 左上から右下へのストローク（太め、先端が尖る）
2. 右上から左下へのストローク（太め、先端が尖る）  
3. 各ストロークの**先端は鋭角に尖っている**（直方体の平らな端ではない）
4. 2本のストロークは中央で交差し、**一体の形状**として見える
5. 全体のシルエットは「𝕏」であり、単なる「×」ではない

### 重要な形状ポイント
- 各バーの端は**台形/楔形**に尖る（上端が狭く、下端が広い、またはその逆）
- ストロークの太さは均一ではなく、中央が太く端が細くなる傾向
- 正確な再現のためには、X社の公式ロゴの SVG パスを参考にすること

## 完了条件

1. Blender 4.x / 5.x で動作するPythonスクリプト
2. 生成されるメッシュが X 社ロゴの正しいシルエットになっている
3. 各ストロークの先端が尖っている（平らな端ではない）
4. マテリアル（黒メタリック）適用済み
5. `.glb` エクスポートコード付き（Three.js で使うため）
6. スクリプト単体で `blender --background --python scripts/blender-x-logo.py` で実行可能

## 実装アプローチ（推奨）

### 方式A: SVG パスからメッシュ生成（推奨）
1. X ロゴの輪郭を頂点座標のリストとして定義
2. `bmesh` で面を生成
3. Solidify（押し出し）で厚みを付ける
4. ベベルでエッジを面取り

### 方式B: 台形ベースの交差
1. 各ストロークを台形（先端が狭い）として生成
2. 結合してブーリアン union
3. 厚み・面取り追加

**方式A を優先。** X ロゴの正確な頂点座標は Web 検索で公式 SVG を参考にしてよい。
